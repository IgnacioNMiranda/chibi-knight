# Chibi Knight

## Description | Descripción

This application is a Discord bot that allows to do simple tasks like send messages or play minigames.

</br>

## Installation | Instalación

```bash
# with npm
npm install
# with yarn
yarn
```

## Running the app | Ejecutando la aplicación

```bash
# development
$ npm run start:dev
# using yarn
$ yarn start:dev
```

## Test

```bash
# unit tests
$ npm run test
# using yarn
yarn test
```
